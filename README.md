Подшаг 2.1. Общие сведения

Подшаг 2.2. Назначение и цели
Система предназначена для автоматизации учета продаж, складских запасов и взаиморасчетов в розничной торговле.

Конкретные цели проекта:

Сокращение времени закрытия смены на 50%.
Повышение точности учета товаров на складе.
Упрощение процесса формирования отчетности.
Подшаг 2.3. Требования к функциям
Система должна обеспечивать выполнение следующих ключевых функций:

Ведение справочников: номенклатуры товаров, контрагентов, подразделений, сотрудников.
Оформление документов продажи: чеки, накладные, счета-фактуры.
Оформление документов поступления: приходные накладные, товаросопроводительные документы.
Формирование отчетов: оборотно-сальдовая ведомость, анализ продаж, отчеты по движениям товаров.
Управление складскими запасами: контроль остатков, инвентаризация, резервирование товара.
Поддержка многопользовательского режима: одновременная работа нескольких пользователей с системой.
